# 小悟AI — 电脑版（零门槛版）

## 你需要做什么？就3步

### 第1步：安装 Python

1. 打开浏览器，访问：**https://www.python.org/downloads/**
2. 点那个 **黄色的 "Download Python 3.x"** 按钮
3. 双击下载好的文件开始安装
4. ⚠️ **最重要的一步：安装界面底部，勾选 "Add Python to PATH"**
5. 点 Install Now，等它装完
6. 装完点 Close

> 就像装一个普通软件一样简单。

### 第2步：把小悟放到电脑上

把 `ai-learner` 文件夹复制到你的电脑桌面就行。

### 第3步：双击运行！

找到文件夹里的 **「小悟-启动.bat」** → 双击它

然后你就能看到小悟在学习了！日记会自动保存在 `diary` 文件夹里。

---

## 每次想看日记

打开 `ai-learner/diary/` 文件夹，里面就是所有学习日记。

## 让小悟每天自动学习

### 方法A：最简单——设个开机自启

右键点击 **「小悟-启动.bat」** → 复制

按 `Win + R` 键 → 输入 `shell:startup` 回车

在弹出的窗口里 → 粘贴（快捷键 Ctrl+V）

这样每次开机就会自动运行一次。如果想要一天跑多次，可以用方法B。

### 方法B：用 Windows 任务计划程序（更灵活）

1. 按 `Win` 键 → 搜索 **"任务计划程序"** → 打开
2. 右侧点 **"创建基本任务"**
3. 名称填：**小悟每日学习**
4. 触发器选：**每天**
5. 时间选：**09:00:00**（可以创建多个任务，比如14:00、21:00各一个）
6. 操作选：**启动程序**
7. 浏览选择：**ai-learner\小悟-启动.bat**
8. 完成！

---

## 不想用电脑开机触发？也行

每次你想让小悟学习时：
1. 双击 **「小悟-启动.bat」**
2. 等30秒
3. 去 diary 文件夹看新日记

就这么简单。
